# User Profile

Adapted from Claude memory for better OpenCode collaboration.

## Technical Profile

- Advanced Linux user across Devuan, Debian, Guix, and Xubuntu
- Prefers literate programming with Org, Babel, and tangling
- Operates a multi-machine setup including ThinkPad X230 and ThinkPad T480s
- Uses Obsidian as the primary knowledge vault
- Uses Spacemacs, NeoVim, and VS Codium for editing
- Uses WezTerm as the current terminal daily driver

## Communication Preferences

- Prefer concise, direct responses
- Avoid unnecessary churn or repeated rework
- Default toward solving the requested problem in one shot
- Treat blunt feedback as precision pressure, not hostility

## Workflow Preferences

- Fix the smallest possible surface area first
- Do not break adjacent behavior while fixing one issue
- Track significant work in the HelmCortex TODO files using Obsidian Tasks format
- Assume late-night, highly iterative terminal work is normal
