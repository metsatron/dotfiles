# Task Tracking

Canonical task tracking lives in HelmCortex's Obsidian vault.

## Task Format

Use Obsidian Tasks plugin syntax:

- `- [ ] Task`
- `- [x] Task ✅ YYYY-MM-DD`

## Progress Convention

When recording completed work, add a dated progress block at the top of the relevant file:

```md
## Progress - YYYY-MM-DD (OpenCode Session)
- [x] What was done ✅ YYYY-MM-DD
- [ ] What remains
```
