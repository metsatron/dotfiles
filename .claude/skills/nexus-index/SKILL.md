# NEXUS Skill & Agent Repos

All repos live at `/home/metsatron/mnt/x230/HelmCortex/NEXUS/git/`. Update with `/nexus-update`.

## awesome-claude-skills

Curated list of Claude Skills, resources, and tools. Not skills themselves — use as a discovery index to find new skills to install. Links to marketplace, API, and community contributions.

## claude-forge

"oh-my-zsh for Claude Code" — a complete development environment plugin with 11 agents, 40 commands, 15 skills, 15 hooks, and 9 rule files.

Subdirectories: `agents/`, `cc-chips/`, `cc-chips-custom/`, `commands/`, `hooks/`, `knowledge/`, `rules/`, `scripts/`, `skills/`, `setup/`, `reference/`, `docs/`

Use for: borrowing agent designs, command patterns, hook implementations, rule templates.

## everything-claude-code

ECC2 framework — performance optimization system for AI agent harnesses. Anthropic hackathon winner. Works across Claude Code, Codex, Cowork.

Subdirectories: `agents/`, `commands/`, `contexts/`, `ecc2/`, `hooks/`, `manifests/`, `mcp-configs/`, `plugins/`, `rules/`, `schemas/`, `scripts/`, `skills/`, `tests/`, `examples/`, `docs/`

Use for: production-ready agent patterns, memory optimization, security scanning, MCP configs, manifest schemas.

## Obsidian-CLI-skill

Standalone Obsidian CLI skill (v1.3.0) covering 130+ commands across files, daily notes, search, properties, tags, tasks, links, plugins, sync, themes, bases, history, and dev tools. Already installed locally as `obsidian-cli` skill.

## obsidian-skills

Kepano's official Obsidian workflow skills following the Agent Skills specification. Contains: `obsidian-markdown`, `obsidian-bases`, `json-canvas`, `obsidian-cli`, `defuddle`. Already installed locally — these are the upstream source.

## skills

Official Anthropic skills repo. Contains example skills for creative/design, development, enterprise, and document skills (docx, pdf, pptx, xlsx). Also contains the Agent Skills specification and skill template.

Use for: reference implementations, the spec itself, document creation skills.
