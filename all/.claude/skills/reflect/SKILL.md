# Session Reflection

Use this when a session has enough durable discoveries to justify a short reflection.

## Steps

1. Identify what was learned.
2. Separate durable knowledge from one-off implementation details.
3. Capture anything worth reusing later.

## Rules

- Keep it brief.
- Prefer durable notes over transcript-style summaries.
