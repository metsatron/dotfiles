# NEXUS Skill and Agent Repos

All repos live at `/home/metsatron/mnt/x230/HelmCortex/NEXUS/git/`. Update with `/nexus-update`.

## awesome-claude-skills

Curated list of Claude skills, resources, and tools. Not skills themselves. Use as a discovery index to find new skills to install.

## claude-forge

"oh-my-zsh for Claude Code" - a complete development environment plugin with agents, commands, skills, hooks, and rule files.

Subdirectories: `agents/`, `cc-chips/`, `cc-chips-custom/`, `commands/`, `hooks/`, `knowledge/`, `rules/`, `scripts/`, `skills/`, `setup/`, `reference/`, `docs/`

Use for: borrowing agent designs, command patterns, hook implementations, and rule templates.

## everything-claude-code

ECC2 framework for AI agent harnesses. Works across Claude Code, Codex, and Cowork.

Subdirectories: `agents/`, `commands/`, `contexts/`, `ecc2/`, `hooks/`, `manifests/`, `mcp-configs/`, `plugins/`, `rules/`, `schemas/`, `scripts/`, `skills/`, `tests/`, `examples/`, `docs/`

Use for: production-ready agent patterns, memory optimization, security scanning, MCP configs, manifest schemas.

## Obsidian-CLI-skill

Standalone Obsidian CLI skill covering note CRUD, daily notes, search, properties, tags, tasks, links, plugins, sync, themes, bases, history, and dev tools. Already installed locally as `obsidian-cli`.

## obsidian-skills

Kepano's official Obsidian workflow skills following the Agent Skills specification. Contains `obsidian-markdown`, `obsidian-bases`, `json-canvas`, `obsidian-cli`, and `defuddle`. Already installed locally.

## skills

Official Anthropic skills repo. Contains example skills for creative, development, enterprise, and document workflows, plus the Agent Skills specification and template.

Use for: reference implementations, the spec itself, and document creation skills.
